## 🔥 Offline Encoding, Decoding & Hashing Toolkit

**Dragonic Decoder** is a complete offline Android application for developers, security enthusiasts, and hackers.

### ✨ Features

#### Encode/Decode (20+)
- Base16/32/58/64/85
- URL, HTML, Unicode
- Binary, Octal, ROT13
- Caesar, Vigenere, XOR
- Morse, JWT, Gzip, Zlib

#### Hashing
- MD5, SHA1, SHA224, SHA256
- SHA384, SHA512, SHA3-256, SHA3-512

#### Tools
- JSON Formatter/Validator
- QR Generator/Decoder
- File Tools (TXT, JSON, CSV, LOG)
- Clipboard Manager
- Password Generator

#### Modern UI
- Dark mode + Cyberpunk theme
- Material Design 3
- Neon green accent
- Bottom navigation
- Search menu
